38

Undo last commit

git reset --soft HEAD~1, git reset --hard HEAD~1

VCS > Git > Reset HEAD... > Select previous commit and choose reset type

 --soft keeps changes staged; --hard discards all changes. Use --hard with caution as it permanently deletes changes.

39

Undo nth commit

git reset --soft HEAD~N, git reset --hard HEAD~N

VCS > Git > Reset HEAD... > Select target commit and choose reset type

 Replace N with the number of commits to undo. --soft preserves changes; --hard discards them. Mixed-intelliJ preserves changes, unstages files

40

Undo all commits

git reset --hard <initial-commit-hash>

VCS > Git > Reset HEAD... > Select initial commit and choose 'hard' reset

 Resets repository to its initial state. Use git log to find the initial commit hash. This operation will delete all commits after the specified commit.
